# My Website

Welcome to my website! This repository contains the source code for my personal website and projects.

## Structure

- `index.html`, `styles.css`, `colors.css`, `fonts.css`: These files define the main structure and styling of the website.
- `assets/`: Contains fonts, icons, and photos used across the website.
- `flappy-react/`: A React-based game project.
- `game/`: Contains additional game-related assets.

## Flappy React Game

The `flappy-react` folder contains the source code for the Flappy React game. If you make any changes to the game, follow these steps:

1. Navigate to the `flappy-react` directory:
   ```bash
   cd flappy-react
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Build the project:
   ```bash
   npm run build
   ```

This will ensure that your changes are properly compiled and ready for deployment.
